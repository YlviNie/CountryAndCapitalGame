A simple Capital guessing game!

Guess the capitals of countries around the world. 

Features:
- Randomize the list of countries so it won't feel repetitive or allow you to memorise the order
- Select countries per continent or select the entire world
- Wrong guesses will automatically be asked again after 5 other questions. This will give you time to learn the countries you aren't very familiar with yet
- Allows you to guess the capitals of countries OR countries to capitals


NOTICE:
This game features 196 countries. The idea for this game is based on the JetPunk website and therefore by the 'Countries of the world' quiz on that website.
